Affinity Development Group

Created a new corporate site for a development firm I worked for. I decided to make everything clean and vibrant with a nice video background on the home page. HTML, CSS, Javascript, jQuery plugins for widgets, carousels, form validation, etc. Simple imagery + iconography, clean typography, responsive design, looks good on mobile/tablet.

http://www.affinitydev.com

GM Microsite

Affinity Development Group's biggest client was Costco Auto, and they wanted a holiday microsite built for General Motors (GM). This site was designed around a few big sections of modern UI and plenty of hero imagery for Chevrolet, Buick, GMC, Cadillac. Used HTML 5, CSS 3, Javascript, jQuery, Bootstrap. I worked in Visual Studio for coding, TFS for source control, and with several very talented .NET developers to create this one. The original site has been archived here.

http://www.costcoautogm.com

Identity Protection Online

This was a large identity protection-related web application that I built most of the front-end for. It's a white-label branded site used by many companies (Geico, Altour, and others). Lots of Javascript, jQuery, HTML, CSS, Sass, responsive design. Worked in BackboneJS framework to produce an excellent user experience.

http://altplus.idprotectiononline.com

Generali Travel Insurance

This is a large e-commerce site I worked on, designed to sell travel insurance with a single-page application feel. I built a lot of the front-end, which was coded on top of an enterprise CMS called Adobe Experience Manager, where I built many of the front-end components and integrated them into AEM, and even used a little React and JSX for a SPA feel throughout the purchase path. Used IntelliJ as my primary IDE for this one.

https://www.generalitravelinsurance.com

Emails

I've coded and designed dozens of emails, based on a few unique brands. Email campaigns can be a fun challenge, because the coding is a fusion of 1998 best practices and modern best practices.
